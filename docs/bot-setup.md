# Инструкция по настройке Telegram-бота

## 1. Создать бота в @BotFather

1. Открыть [@BotFather](https://t.me/BotFather) в Telegram
2. Отправить `/newbot`
3. Ввести отображаемое имя бота (например, `Crypto Magnate Feedback`)
4. Ввести username бота (например, `crypto_magnate_feedback_bot`)
5. Сохранить выданный токен — он понадобится для деплоя бэкенда

## 2. Настроить описание и брендинг

```
/setdescription  → выбрать бота → ввести описание (показывается до /start)
/setabouttext    → выбрать бота → ввести краткое описание (в профиле бота)
/setuserpic      → выбрать бота → загрузить аватар
```

## 3. Подключить Mini App

Вариант A — через `/newapp`:
1. Отправить `/newapp` → выбрать бота
2. Ввести название приложения
3. Загрузить иконку (640×360 px, JPG или PNG)
4. Вставить URL задеплоенного фронтенда (Vercel)
5. Указать короткое имя приложения (используется в ссылке `t.me/<bot>/<shortname>`)

Вариант B — через кнопку меню (`/setmenubutton`):
1. Отправить `/setmenubutton` → выбрать бота
2. Вставить URL фронтенда
3. Ввести текст кнопки (например, `Отправить фидбек`)

## 4. Проверить

1. Открыть бота в Telegram
2. Нажать кнопку меню (или отправить `/start` и нажать кнопку)
3. Убедиться что Mini App открывается и корректно отображается

## 5. Получить Asana refresh token (одноразово)

После регистрации OAuth-приложения в Asana (app.asana.com → Developer Console):

```bash
cd backend
python get_token.py
```

Скрипт откроет браузер, попросит разрешить доступ и выдаст `REFRESH_TOKEN`.
Прописать в `backend/.env`:

```
ASANA_CLIENT_ID=...
ASANA_CLIENT_SECRET=...
ASANA_REFRESH_TOKEN=...
```
